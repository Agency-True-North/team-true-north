What you're installing

A "skill" is a small instruction file that lives on your computer. Claude Code reads it automatically and learns our whole Viral to Script system — the one rule, the kill list, the voice check, both script formats, the tracker. You install it once. It follows you into every project after that.

Everything you need to copy is in a gold-labeled box. Copy the whole box, nothing else.

01 / Before You Start

  • Claude Code is installed and working. If it isn't, do the Claude Setup Guide first, then come back here.
  • You've read the system guide once — Viral to Script System. The skill runs the system, but you should know what it's doing and why.
  • SortFeed is set up in Chrome — that's section 01 of the system guide. It's how you'll gather the batches the skill turns into scripts.

02 / Step 1 — Open Claude Code

Step 1 Open a Claude Code session

Open Claude Code the way you normally do — the terminal, the desktop app, or the VS Code extension. Any folder is fine. You just need a chat box you can type into.

03 / Step 2 — Paste the Install Message

Step 2 Copy the box below and paste it into Claude Code

Copy everything inside the gold-labeled box — the whole thing, exactly as written. Paste it into Claude Code and press enter. Claude will download the skill file and put it in the right place. If Claude asks permission to create the file or fetch the link, approve it.

Copy and paste this into Claude Code
Create the file ~/.claude/skills/viral-to-script/SKILL.md with the exact contents of https://raw.githubusercontent.com/Agency-True-North/team-true-north/main/skills/viral-to-script/SKILL.md — download it, don't rewrite it. When you're done, show me the first 10 lines of the file so I can confirm it worked.
Works on Mac and Windows

Claude figures out the right folder on your machine. You don't need to know where it is — the message above handles it on either system.

Comfortable in the terminal? (Optional — Mac only)

This one-line command does the same thing without Claude's help. Skip it if Step 2 already worked:

Optional — Mac Terminal Alternative
mkdir -p ~/.claude/skills/viral-to-script && curl -fsSL -o ~/.claude/skills/viral-to-script/SKILL.md "https://raw.githubusercontent.com/Agency-True-North/team-true-north/main/skills/viral-to-script/SKILL.md"

04 / Step 3 — Restart Claude Code

Step 3 Close the session and start a fresh one

Skills load when a session starts, so the one you just installed isn't visible to the session that installed it. End the current session and open a new one. That's the whole step.

05 / Step 4 — Check It Took

Step 4 Ask Claude what skills it has

In the new session, paste the question below. You're looking for viral-to-script in the answer. If it's there, you're installed.

Copy and paste this into Claude Code
What skills do you have available? Is viral-to-script one of them?

06 / Step 5 — Run Your First Batch

Step 5 Hand Claude your SortFeed notes

Run SortFeed the way the system guide shows, pick your top reels, and paste them into Claude Code using the shape below. One line per reel: source page, link, view count, and the hook. Add the format if you caught it — it helps.

Copy, fill in your reels, then paste into Claude Code
Turn these virals into scripts: 1. @sourcepage — [link] — 2.1M views — hook: "[the first line of the reel]" — format: [talking head list / b-roll with text / voiceover story] 2. @sourcepage — [link] — 840K views — hook: "[...]" — format: [...] 3. @sourcepage — [link] — 1.4M views — hook: "[...]" — format: [...]

Claude will extract the mechanics, screen out anything from the kill list, and hand back tagged VO and TH scripts — hooks, beats, shot lists, captions, and tracker rows. You make the final call on which to shoot.

07 / If Something Goes Wrong

  • The skill isn't in the list at Step 4. Make sure you started a new session after installing — skills don't appear in the session that created them. If it's still missing, re-run Step 2 and check that Claude confirms the file was created.
  • Claude says it can't reach the link. Check your internet connection and paste the Step 2 message again — the link must stay on one line, exactly as written.
  • Claude wrote its own version instead of downloading. Re-run Step 2 — the message says "download it, don't rewrite it" for exactly this reason. The file should start with three dashes and name: viral-to-script.
  • Still stuck. Bring it to the weekly call — two minutes of screen share solves what an hour of guessing won't.

08 / Updating the Skill Later

When the system improves, the master file on GitHub gets updated — but your copy doesn't update itself. When you hear on a team call that the skill changed, just run Step 2 again. Same message, same box. It overwrites the old version and you're current.

The point of all this

Ten minutes of setup buys you a script partner that already knows the system — the one rule, the kill list, the voice, the woman we serve. Your job stays the human part: pick the virals, film your real life, and make the call on what ships.
