01 / Get It — Download, Unzip, Drag

Download Save the kit

Click the button. Your browser saves Find Skills Skill.zip. No account, no login — the kit is built right into this page.

⇩  Download the Skill (.zip)
If nothing happens, right-click the button → “Save Link As…”
Unzip Open it up

Double-click the downloaded file. You get a folder with a .claude folder and a README inside.

Drag Drop it into your business folder

Drag the whole .claude folder into your Claude folder — the one you open in Claude Code. Don't open it, don't rearrange it. Claude handles the rest.

Already have a .claude folder? Good — just let it merge. The find-skills folder slots in next to your other skills.

02 / Turn It On — Paste One Message

Open your business folder in Claude Code and paste this:

Paste into Claude Code
Is there a skill for [the thing you're trying to do]? Use the find-skills skill.

Claude searches the open library, checks how many people trust each option, and recommends one — with the source and the install command. It only installs when you say go.

03 / How To Use It Every Day

Any time you're about to ask Claude to do something new, ask this first:

  • “Is there a skill for editing my videos?”
  • “Find me a skill that turns a doc into a slide deck.”
  • “Can you do PDF form-filling? Check for a skill first.”
What it checks before it recommends

How many people use the skill (it prefers the ones thousands trust), and who made it (official sources over unknowns). It'll warn you off anything unproven instead of installing it blind.

It checks your own skills first

Most of what you do every day already has a skill you built. This one only goes shopping when you need something genuinely new.

04 / No Claude Code Yet?

Get the Claude desktop app (Mac or Windows) — the non-technical-friendly version, no terminal required. Sign in, open your business folder, and paste the message above. If you prefer the command line, the CLI works the same way.

05 / Where The Skill Lives

The kit puts it here:

Folder map
your-business-folder/ └── .claude/ └── skills/ └── find-skills/ └── SKILL.md

Skills in a project's .claude/skills/ load whenever you open that folder in Claude Code. To make it available in every project on your machine, drag the find-skills folder into ~/.claude/skills/ instead.

06 / Good To Know

  • This finds general-purpose skills from the wider community. They won't know your brand voice or your funnel — keep using your own custom skills for anything brand- or content-facing.
  • Reach outside only for a genuinely new capability — a file format, a dev tool, a new integration.
  • Nothing installs without your say-so. Claude shows you the option and the source first, every time.
